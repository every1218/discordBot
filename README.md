# 펭귄봇 명령어 목록

이 문서는 펭귄봇에서 사용 가능한 명령어들을 정리한 것입니다.

## 📅 일반 명령어 (`commands.py`)

- `.준엽 전역`: 준엽의 전역까지 남은 날짜를 보여줍니다.
- `.정혁 전역`: 정혁의 전역까지 남은 날짜를 보여줍니다.
- `.도현 입대`: 도현의 입대까지 남은 날짜를 보여줍니다.
- `.상훈 입대`: 상훈의 입대까지 남은 날짜를 보여줍니다.
- `.수능`: 수능까지 남은 날짜를 보여줍니다.
- `.4수`: 다음 수능까지 남은 날짜를 보여줍니다.
- `.점메추`: 점심 메뉴를 랜덤으로 추천해줍니다.
- `.라인`: 리그 오브 레전드 라인을 랜덤으로 추천해줍니다.

## 💰 경제 명령어 (`economy.py`)

- `.돈`: 현재 자신의 자산과 레벨을 확인합니다. 
- `.시급`: 1시간마다 일정량의 돈을 받습니다.
- `.분급`: 1분마다 일정량의 돈을 받습니다.
- `.출석`: 하루에 한 번 출석하여 레벨을 올립니다. (레벨이 높을 수록 시급&분급이 늘어납니다)
- `.순위`: 서버 내 유저들의 돈 순위를 보여줍니다.
- `.송금 [금액] [유저]`: 다른 유저에게 돈을 보냅니다. (예: `.송금 1000 @유저이름`)

## 🎮 게임 명령어 (`games.py`)

- `.홀짝 [홀/짝] [금액]`: 주사위의 결과가 홀수인지 짝수인지 맞추는 게임입니다. (예: `.홀짝 홀 10000`)
- `.잭팟`: 슬롯머신을 돌려 잭팟을 노립니다. ()
- `.복권 [숫자1] [숫자2] [숫자3] [숫자4] [숫자5] [숫자6]`: 1부터 20까지의 숫자 6개를 맞춰 복권에 당첨됩니다. (예: `.복권 1 2 3 4 5 6`)
- `.블랙잭 [베팅금액]`: 딜러와 블랙잭 게임을 합니다. (예: `.블랙잭 50000`)

## ⚔️ 롤 카운터 명령어 (`champion.py`)

- `.카운터 [챔피언 이름]`: 해당 챔피언의 카운터 정보를 보여줍니다. (예: `.카운터 가렌`)

## 💬 AI 채팅 명령어 (`chat.py`)
- `@봇이름 [할 말]`: 봇을 멘션하여 인공지능과 대화할 수 있습니다. (예: `@펭귄봇 ㅎㅇ`)
